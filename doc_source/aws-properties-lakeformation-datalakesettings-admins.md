# AWS::LakeFormation::DataLakeSettings Admins<a name="aws-properties-lakeformation-datalakesettings-admins"></a>

A list of AWS Lake Formation principals\.

## Supported Regions

This PropertyType is supported by the following regions:

- `ap-northeast-1`
- `eu-west-1`
- `us-east-1`
- `us-east-2`
- `us-west-2`
