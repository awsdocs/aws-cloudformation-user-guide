```
{
  "[VisibilityState](#cfn-quicksight-dashboard-sheetcontrolsoption-visibilitystate)" : String
}
```

### YAML<a name="aws-properties-quicksight-dashboard-sheetcontrolsoption-syntax.yaml"></a>

```
  [VisibilityState](#cfn-quicksight-dashboard-sheetcontrolsoption-visibilitystate): String
```

Visibility state\.  
*Required*: No  
*Type*: String  
*Allowed values*: `COLLAPSED | EXPANDED`
